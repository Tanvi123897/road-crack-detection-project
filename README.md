# Road Crack Detection Project

## 📌 Overview
This project is a simple road crack detection system using Python and image processing techniques.

## ⚙️ Technologies Used
- Python
- OpenCV
- Jupyter Notebook 

## 🧠 Method
- Load road image
- Convert to grayscale
- Apply edge detection using Canny algorithm
- Highlight cracks in the image

## 📂 Files
crack_detection.py → Main code
asphalt-highway.jpg → Input image

## 🎯 Purpose
To detect cracks in road images automatically and support infrastructure maintenance.
